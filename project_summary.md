# 專案總覽：圖片轉 GIF 工具 (GIF Converter)

這是一份關於「圖片轉 GIF 工具」專案的目前狀態、功能清單與架構摘要，記錄截至 2026-04-23 為止的所有進度。

## 📁 檔案與目錄架構
本地端開發路徑：`i:\我的雲端硬碟\粉專\008 數位專案與互動程式\1150422圖片傳GIF\`

專案為純前端 (Vanilla HTML/CSS/JS) 架構，所有處理皆在使用者瀏覽器端完成，包含三個核心檔案：
1. **[index.html](file:///i:/%E6%88%91%E7%9A%84%E9%9B%B2%E7%AB%AF%E7%A1%AC%E7%A2%9F/%E7%B2%89%E5%B0%88/008%20%E6%95%B8%E4%BD%8D%E5%B0%88%E6%A1%88%E8%88%87%E4%BA%92%E5%8B%95%E7%A8%8B%E5%BC%8F/1150422%E5%9C%96%E7%89%87%E5%82%B3GIF/index.html)**：網頁主結構、UI 元件佈局與 CDN 外部套件引入。包含奕鈞老師製作的版權宣告與版本 1.3 資訊。
2. **[style.css](file:///i:/%E6%88%91%E7%9A%84%E9%9B%B2%E7%AB%AF%E7%A1%AC%E7%A2%9F/%E7%B2%89%E5%B0%88/008%20%E6%95%B8%E4%BD%8D%E5%B0%88%E6%A1%88%E8%88%87%E4%BA%92%E5%8B%95%E7%A8%8B%E5%BC%8F/1150422%E5%9C%96%E7%89%87%E5%82%B3GIF/style.css)**：專案樣式表。包含毛玻璃效果介面、動態背景、拖曳排序視覺反饋與文字設定面板樣式。
3. **[script.js](file:///i:/%E6%88%91%E7%9A%84%E9%9B%B2%E7%AB%AF%E7%A1%AC%E7%A2%9F/%E7%B2%89%E5%B0%88/008%20%E6%95%B8%E4%BD%8D%E5%B0%88%E6%A1%88%E8%88%87%E4%BA%92%E5%8B%95%E7%A8%8B%E5%BC%8F/1150422%E5%9C%96%E7%89%87%E5%82%B3GIF/script.js)**：負責圖片上傳、拖曳排序、單張圖片文字/樣式客製化、裁切互動 (Cropper.js)、以及 GIF 生成邏輯。

## ✨ 核心功能清單

### 1. **流暢的圖片管理系統**
* **拖曳排序 (Drag and Drop)**：上傳後可直接滑鼠拖曳縮圖調整播放順序，編號會自動更新。
* **即時尺寸顯示**：縮圖左下角即時顯示圖片（或裁切後）的解析度，無須點擊即可掌握。
* **刪除與管理**：支援單張刪除或全數清除。

### 2. **單張圖片文字客製化**
* **文字輸入**：每張圖片下方可獨立輸入要疊加的文字。
* **進階樣式設定**：點擊「文字樣式設定」可調整：
    * **字體選擇**：支援 Inter, Noto Sans TC 等多種字體。
    * **字體大小**：自訂像素大小。
    * **渲染背景**：可開啟/關閉文字背後的圓邊長方形背景。
    * **漸層背景**：內建五組現代感漸層（晨曦紫、深海藍等）。
    * **透明度調整**：自訂背景條的透明度。

### 3. **裁切與設定**
* **指定比例裁切**：強迫鎖定於 GIF 比例進行裁切。
* **全域參數**：每張圖每幀秒數、輸出寬度與比例選擇。

### 4. **客戶端直接產生動畫 (無須後端)**
* 整合 `gifshot.js`，將 Canvas 處理後的每一幀合建成高品質 GIF，具備進度條與下載功能。
